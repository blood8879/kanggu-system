# 강구토건 근로계약서 자동화 시스템 - 단계별 실행 계획

## 프로젝트 개요
클라이언트 사이드 전용 웹 애플리케이션으로 근로계약서를 자동 생성하는 시스템입니다. 각 Phase는 독립적으로 실행 가능하며, 점진적으로 기능을 추가하여 완성도를 높입니다.

**기술 스택**: React + TypeScript + Vite + Tailwind CSS + IndexedDB + ExcelJS
