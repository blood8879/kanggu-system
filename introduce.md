너는 지금부터 PRD 작성 전문가인 프로덕트 매니저야. 아래 용소를 포함한 PRD를 작성해줘.

# 문제 정의

(주)강구토건에서는 시규 일용직 근로자가 올때마다 근로계약서를 작성하는데, 기존 출근하던 일용직 근로자들도 매달 1일이 되면 계약서를 신규 작성해야해. 그런데 신규 근로계약서를 작성할 때 이전에 출근 이력이 있는 근로자인 경우 근로자가 수기로 작성해야 할 항목수를 더 줄이기 위해 웹 ui에서 간단한 인적사항을 기록하면 자동으로 근로계약서 형식을 만들어주는 시스템을 구축하려 해.

# 타겟 사용자

반복적인 서류 업무를 하는 근로자들을 대상으로 할거야.

# 제안 해결책

타겟 사용자들이 쉽고 직관적으로 웹 ui상에서 필요한 정보만 입력하면 해당 정보가 모두 기입된 엑셀 파일을 생성해서 다운로드 가능하게 해주고, 사용자는 파일만 다운로드 해서 출력하면 되도록 할거야. 

# 목표 및 성공 지표

근로 계약서 자동화를 시작으로 추후에도 자동화가 필요한 서류의 양식을 가지고 생성할 수 있도록 할거야.
